Improvements

Written 18/12/2020

As mentioned before I feel like I could review my use of GitHub, while I feel like my commits were good and overtime the titles improved I do think I could have used branches to tidy up my commits. This would be my main improvement if I was to do this project again.

If I had longer to work on this project, I would consider addinng personal images and graphics to my project to personalise it a bit more to myself and to avoid any risk of copyright issues, using personal issues would also help improve the professionalism of the site.

In the future I will need to update this site and add more content to it. I could see the CV page and the Projects page needing updating frequently as I make more projects over the course of my career as well as gaining more work related experience.

I could consider adding additional style sheets to my work, especially a stylesheet for a printable version of the website. My current website is not very printableand would waste a lot of ink to print it out, it could be handy to have a printable version in case I wanted to use it in a paperbased portfolio, or if anyone using my portfolio wanedto be able to print my work history.

I feel like I could improve my site by making it more accessible. I feel like it currretly my website is screen reader friendly as I have used alternate text for images which screen readers can read and I split sections up to stop larage chunks all being read at once. I could make my site more accessible in the future by adding a stylesheet for people who are colour blind, giving a "large mode" to increae font size for people who struggle to read smaller text as well as possibly offering more language choices

Currently my website uses the default Icon on the tab bar. I feel like this is boring and doesn't make it stand out, it also means that when the site is pinned in a browser you don't have a way to tell what site is pinned without clicking on it. I feel like adding icon would do a lot to make my site look more professional.

I could improve the impressive effect my site could have be adding more animation, a lot of the sites I saw during personal research has a lot of animation on them to wow the user and to make the sites memorable. However it is important to remember there is animation limitatins within CSS and personal limitaitons due to a knowledge gap that I have. If I was to add more animations it would be important to consider site performance as too many animations could make the site unresponsive for some users which can be frustrating

To make my site more interesting and show a better technical ability I could add videos to the site. At the momement I dont have any work that jusitifies adding a video but in the future I will, the code for adding a video is slightly different to the code I have used so far so it would show a larger range in my work.
